# dtpinardumancapraz Web Sitesi

## Kurulum

1. **WhatsApp numarası:** `script.js` dosyasında `WHATSAPP_NUMBER` değişkenini kendi numaranızla değiştirin.  
   Örnek: `905321234567` (ülke kodu 90, başındaki 0 olmadan)

2. **BEFORE / AFTER görselleri:**  
   `images/` klasörüne aşağıdaki isimlerle fotoğraflarınızı koyun:
   - `before1.jpg`, `after1.jpg` (İş #1)
   - `before2.jpg`, `after2.jpg` (İş #2)
   - `before3.jpg`, `after3.jpg` (İş #3)  
   Daha fazla iş eklemek için `index.html` içindeki `ba-grid` bölümüne yeni `.ba-card` kopyalayabilirsiniz.

3. **Siteyi açmak:** `index.html` dosyasını tarayıcıda açın veya bir yerel sunucu kullanın.

## Özellikler

- Ana sayfa BEFORE – AFTER çalışmalar
- Sabit WhatsApp iletişim butonu (sağ alt)
- İletişim bölümünde WhatsApp butonu ve numara
